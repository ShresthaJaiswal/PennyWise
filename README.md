## Project overview

Individuals frequently struggle to manage their personal finances due to fragmented tracking, lack of real-time budget visibility, and tedious manual entry systems. PennyWise addresses this by providing a unified, high-performance web application that simplifies financial logging. By combining dynamic data tables, streamlined validation forms, and interactive drag-and-drop budgeting tools, the application eliminates friction in expense categorization and empowers users to make informed, data-driven financial decisions.


## Technical Stack & Documentation
- UI Framework: React
- Global State: Redux Toolkit
- Data Fetching & Caching: RTK Query
- Data Grid & Ledger: TanStack Table (React Table)
- Form Management: React Hook Form
- Component Library: Material UI (MUI)
